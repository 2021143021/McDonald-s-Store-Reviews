## 📌 프로젝트 개요

햄버거를 정말 좋아한다. 특히 맥도날드는 빠르고 저렴한 가격에 전 세계적으로 사랑받는 브랜드다.  
하지만 모두가 맥도날드를 긍정적으로 생각할까?

이번 프로젝트에서는 전 세계 매장에 남겨진 리뷰를 수집해서, **사람들이 실제로 맥도날드에 대해 어떻게 느끼는지** 감정 분석 모델로 분석해봤다.

> 자연어처리(NLP) 모델인 **MobileBERT**를 기반으로 리뷰 텍스트를 긍정/부정으로 분류하는 감정 분석을 수행했다.

---

## 📁 데이터 정보

| 구분 | 수치 |
|------|------|
| 전체 리뷰 수 | 33,396건 |
| 사용한 최종 데이터 수 | 28,570건 |
| 학습:검증 비율 | 8:2 |
| 중립(3점) 제거 후 남은 감정 라벨 | 1, 2점 → 부정 / 4, 5점 → 긍정 |

### 📊 리뷰 평점 분포

- 점수는 1~5점 (1~2: 부정, 4~5: 긍정, 3: 중립)
- 3점은 감정 중립적이기 때문에 라벨링에서 제외했다

## 🔍 라벨링 기준

- 평점 1, 2 → 부정 리뷰 (`label = 0`)
- 평점 4, 5 → 긍정 리뷰 (`label = 1`)
- 평점 3점 중립 → 제외 (모델이 긍/부정 이분법으로 학습되도록 하기 위해)

## 📌 결론 및 느낀 점

- MobileBERT는 짧고 비정형적인 리뷰 텍스트에도 강한 성능을 보였다
- 학습량이 많을수록 모델 성능이 향상됐고, 28,000건 학습에서는 검증 정확도 **95%**에 도달했다
- 중립 라벨 제거로 모델은 단순해졌지만, **긍/부정 경계에 있는 데이터는 배제**됐다 → 향후 개선 필요
- 평점과 텍스트 감정이 항상 일치하지는 않았다 → 감정 기반 평점 조정/보정의 가능성이 있다
